# 🎬 GKodi

> PowerShell script to **download, install, and configure Kodi** with addon repositories (The Crew, Venom, Seren).

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📥 **Download** | Fetches Kodi 20.2 Nexus x64 from mirrors.kodi.tv |
| ⚙️ **Silent Install** | `/S` installer |
| 🌐 **Web Server** | Enables Kodi web server on port 8080 (user: kodi, pass: kodi) |
| 📦 **Repositories** | Adds The Crew, Venom, Seren sources |
| 📋 **JSON-RPC** | Installs addons via Kodi JSON-RPC API |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 (64-bit) |
| **PowerShell** | 5.1+ |
| **Network** | Internet for download |

---

## 🚀 Usage

```powershell
# Run — downloads, installs, configures
.\GKodi.ps1
```
